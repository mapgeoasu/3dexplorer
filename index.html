About the 3D Explorer

Welcome to the virtual Map and Geospatial Hub at the Arizona State University Library!

Find us in the real world in ASU’s Hayden Library, Tempe, Arizona, USA.



Here at the Hub, we hold thousands of maps, aerial photographs, and other geographic information resources.

Use this 3D Explorer application to virtually explore the Hub space and its resources.

Navigate the Scene

Use the navigation controls to move through and reposition the interactive 3D map scene: zoom-in, zoom-out, pan, rotate, orient north.



Use the “home” button to return to the default scene view. Use the “play” button to auto-rotate the scene; “pause” stops the rotation.

Explore the Hub Space

More information is available for nearly every object in the 3D model. Explore the Hub by selecting features in the space.

For example, you can select any map cabinet drawer (or other storage features, such as shelves) to get a basic summary of the contents stored in that particular location.



You can also select office locations to learn more about the staff members who occupy them, as well as how to get in touch.



Or select globes to learn about and view photos of each one.



Select other features in the workspace too, such as equipment and furniture.



Search for Resources by Attribute

Search for maps, aerial photos, and other geographic information resources by attributes such as keyword or call number.



Use Advanced Search to apply multiple criteria to your search, such as format, theme, region, timeframe, etc.



Item-level search results will appear in the sidebar. Throughout the scene, all locations containing the search results will be highlighted.



The sidebar and pop-up box will indicate the total number of items identified by your search. The sidebar also groups and counts search results by each location.

Find Resources by Location

You can find resources by location too. Simply select any storage feature (drawer or shelf). The pop-up box will summarize the contents of that location.



Select the “View Item List” button in the pop-up box to view a list of all items contained in that location.



View Resource Details

In the sidebar, you’ll see a list of all items contained in a given location. Select any of the items in the sidebar to learn more details about it.



The pop-up box will reveal details such as the item’s title, publication date, author, publisher, cartographic scale, and library catalog record.

When available, you will also find a link to view a particular item in a separate spatial index web map, or a link that describes the item in deeper context.

A thumbnail image of the item will also appear in the pop-up box when available.

View Resource Image

If a thumbnail image is present, you can select it to view it more closely. Use the image viewer controls to rotate, flip, or otherwise manipulate the view of the image.



View the Hub Space

Select the button with the image icon to view a 360 degree panorama photo of the space. Navigate within the panoramic viewer to move between images of the Hub interior and exterior.
